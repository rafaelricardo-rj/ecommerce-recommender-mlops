## 📌 Project Overview

End-to-end MLOps pipeline for e-commerce product recommendations. Features a robust infrastructure for reproducible ML, integrating DVC + MLflow for model governance and Kubernetes for container orchestration. Codebase focuses on modularity and Clean Code patterns, ensuring a scalable and maintainable AI service.

## 🛠 Tech Stack

- **Language:** Python 3.11+
- **Deep Learning Framework:** PyTorch
- **Dependency Management:** Poetry / uv
- **Data Versioning:** DVC (Data Version Control)
- **Experiment Tracking & Registry:** MLflow
- **Containerization:** Docker
- **Orchestration:** Kubernetes
- **Architecture:** Clean Architecture & SOLID Principles
